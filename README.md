## 使用

1. 先安装OpenCC模块：

   ```
   pip install opencc-python-reimplemented
   ```
   
2. 然后下载 `converter.py`
   
2. 选择（可以多选）epub文件，拖到 `converter.py` 上放开；或者直接打开 `converter.py` 选择（可以多选）epub文件
   
2. 在打开的命令行窗口输入选择的转换对应的数字，按 `Enter`
   
2. 等待完成
